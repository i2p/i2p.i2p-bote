Ce este o Identitate?

O identitate este ca un cont de e-mail, dar fără un furnizor - vă controlați contul, în locul unui terț precum Google sau Yahoo.

Ce este o destinație?

O destinație este adresa pe care o oferiți prietenilor, astfel încât aceștia să vă poată trimite e-mailuri.

De ce este adresa de email așa de lungă?

Fiecare e-mail este criptat automat de către Bote. Fiecare adresă conține un set de chei publice, care sunt utilizate pentru criptarea e-mailurilor trimise la adresa respectivă. Lungimea adresei depinde de tipul de criptare utilizat.

Există o limită a numărului de identități pe care le pot avea?

Puteți creea cât de multe identități doriți și puteți utiliza diferite identități pentru diferite scopuri.

Când expiră identitățile?

Nu. Atâta timp cât ai cheile private pentru o identitate, o poți folosi. Acest lucru înseamnă, de asemenea, că nu există "recuperare de cont" - dacă pierdeți telefonul, vă pierdeți Identitățile.

Puteți face backup pentru identitățile dvs., exportându-le din meniul „Setări”. Acest lucru va salva o copie a acestora într-un fișier din stocarea telefonului. Fișierul poate fi folosit pentru a recupera identitățile, sau pentru a le muta pe un alt dispozitiv.
