Débuter

Il vous faut d’abord une identité. Créez-en une dans le menu « Paramètres » ou importez une identité existante. Ensuite, vous pourrez ajouter les adresses de vos amis par copier-coller ou les échanger par codes QR ou NFC.

Sur les versions d’Android antérieure à 4.4, il est recommandé d’installer le gestionnaire de fichiers « OI File Manager » pour améliorer la sélection des fichiers. Pour partager par codes QR, installez le lecteur de codes QR « Barcode Scanner ». Cliquer sur les liens ouvrira les logithèques Google Play Store ou F-Droid pour permettre l’installation.
